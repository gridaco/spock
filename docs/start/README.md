# Getting started

These pages take you from nothing to a running Spock backend: install the
toolchain, scaffold and serve a first project, then build a small product
backend from scratch.

- [Install](install.md) — the npm package, supported platforms, and version
  requirements.
- [Quickstart](quickstart.md) — scaffold a project, declare two tables, and
  read, write, and break things against the derived API in about ten minutes.
- [Tutorial: build a mini-Instagram backend](tutorial.md) — a complete
  four-table authority with seed personas, derived errors, functions, and a
  refusal.
