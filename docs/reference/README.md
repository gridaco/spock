# Reference overview

Lookup pages for the surfaces you touch while operating Spock. Each page is a
projection of the [v0 specification](../spec/v0.md) or of shipped toolchain
behavior, organized for retrieval rather than reading order.

- [CLI](cli.md) — every `spock` command, flag, and default.
- [HTTP API](http.md) — endpoints, REST operators, GraphQL binding, and the
  storage plane.
- [Error codes](errors.md) — the runtime error vocabulary and compile-time
  diagnostics.
- [spock.toml](spock-toml.md) — the framework project manifest.
